## 🚀 로컬 실행 방법 (Quick Start)

> ⚠️ 주의: `node_modules` 폴더는 제외되어 있으므로, 최초 실행 시 반드시 **패키지 설치(Install)** 작업을 먼저 진행해야 합니다.


## 🛠️ 시작하기 (How to Start)

이 프로젝트를 로컬 환경에서 실행하려면 아래 순서대로 명령어를 입력해 주세요.

### 1. 프로젝트 복제 (Clone)
```bash
git clone [https://github.com/jungsuryeon/BREDRED_test.git](https://github.com/jungsuryeon/BREDRED_test.git)
cd BREDRED_test
```

### 2. 의존성 라이브러리 설치
```bash
# npm을 사용하는 경우
npm install

# 만약 pnpm을 사용하시는 경우
pnpm install
```

### 3. 로컬 서버 실행
```bash
npm run dev
# 또는 pnpm dev
```

### 서버가 켜지면 터미널에 뜨는 http://localhost:1234 같은 주소로 접속
